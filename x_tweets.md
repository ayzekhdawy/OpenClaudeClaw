## Seçenek 4: Teknik Detay Sevenler İçin

OpenClaudeClaw Technical Deep Dive 🧵

1/ Harness Architecture:
- Runtime (HarnessRuntime)
- Tool Pool (42 tools registered)
- Context Builder (auto-merge SOUL/MEMORY/USER)
- Policy Engine (permission rules + approval)
- State Machine (.harness/*.json persistent)

2/ Tool Categories:
Core: Bash, Read, Write, Edit, Glob, Grep, Think, Task
Extended: WebSearch, WebFetch, TodoWrite, Brief, Task CRUD...
Advanced: LSP (AST-based), REPL, PlanMode(4), Worktree(3)...

3/ Performance:
- Import: 0.6s
- Pool init: 0.6s
- Tool exec: 0.0-0.7s
- Cache: 60s TTL (0ms repeated queries)

4/ Installation:
Selective install wizard:
- Core only (8 tools)
- Core + Extended (27 tools)
- All (42 tools)

Custom state dir, policy config, cache settings.

5/ Code Quality:
- Type hints throughout
- Test suite included
- 3 demo workflows
- Full documentation

MIT licensed. Production ready.
